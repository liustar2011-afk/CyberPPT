强制切换：Scene Graph First

所有脚本生图转 PPT 必须先通过 page_scene_graph_gate；不通过则不导出 PPTX。

Gate 分层

Truth Gate
文本必须来自脚本/内容锁/人工覆盖；禁止 OCR 改写。
Geometry Gate
坐标空间必须可解释；容器、安全区、text zone 均需归一化。
Layout Gate
文本必须绑定容器并避让图标、箭头、分隔线。
Render Gate
导出后截图检查可见性、裁切、重叠、文本缺失。
Capture Gate
source_capture 必须记录 scene graph、关系、QA 结论。

失败不再静默

允许继续

  • 所有文本节点有 truth source。
  • 每个文本节点有 bound_container_id。
  • 容器有 text_safe_bbox 与 reserved zones。
  • 渲染后文本完整且可见。

阻断导出

  • 只有 OCR bbox，没有语义容器。
  • registry 有容器但未绑定文本。
  • 坐标归一化存在未解释漂移。
  • 源脚本和可编辑文本不一致。
