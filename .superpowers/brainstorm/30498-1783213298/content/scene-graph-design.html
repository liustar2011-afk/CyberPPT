硬性 Gate

必须通过

  • 所有文本节点必须来自脚本 truth 或明确人工覆盖。
  • 所有文本节点必须绑定容器，不能只有 OCR bbox。
  • 所有容器必须有 normalized bbox 和 text_safe_bbox。
  • 文本 bbox 不能出容器，不能压图标、箭头、分隔线。
  • 源图 text zone / icon / container 关系必须进入 source_capture。

必须阻断

  • 坐标系不一致但没有 transform 解释。
  • registry 有承载容器但无对应文本节点。
  • 脚本 truth 与语义计划文本不一致。
  • render 后文字不可见、重叠、越界或被裁切。
