# Stage 2 Visual Lock - Style 04

Selected style: **04 象牙白 + 深蓝强调**.

Scope: P6-P9 only.

## Palette

| Use | Color |
|---|---|
| Page background | `#F7F6F0` |
| Title / primary text | `#101820` |
| Body text | `#303030` |
| Secondary text | `#6F7275` |
| Divider / line | `#C9CDD1` |
| Accent | `#12355B` |

## Tone

Formal enterprise digitalization and platform-architecture style. This direction is suitable for P6-P9 because the selected pages emphasize overall architecture, closed-loop operation, evaluation model, and scenario services.

## Geometry And Grid

- Canvas: 16:9.
- Body blueprint region only; do not draw slide title, subtitle, logo, page number, footer, master red/blue lines, or full PPT frame.
- Use a restrained 12-column business grid.
- Prefer architecture maps, process lanes, matrices, tiered stacks, sidebars, and rule-based panels.
- Keep high information density from the Stage 1 page plan; do not simplify P6-P9 into sparse cards.

## Surface System

- Use ivory continuous paper background `#F7F6F0`.
- Main panels use subtle off-white or cool-light-gray surface shifts, not heavy white card stacks.
- Divide dense regions with fine `#C9CDD1` rules and thin section headers.
- Use deep blue `#12355B` for primary anchors: conclusion bars, key path nodes, core system blocks, active arrows, and strategic emphasis.
- Avoid heavy shadows, decorative gradients, glassmorphism, bright neon, or dashboard-like dark surfaces.

## Typography

- Target language: Simplified Chinese.
- Language source: source material.
- Use modern Chinese sans-serif visual language, close to Source Han Sans / PingFang SC / Microsoft YaHei.
- Use the repository fixed Typography Scale `C0` + `T1-T14`; do not invent new levels.
- Body blueprint text is visual placeholder only; final PPT text must come from content locks.

## Chart And Diagram Language

- P6: high-density architecture map with source column, core platform block, object/evidence pool, application column, service bar, and compliance spine.
- P7: closed-loop operating chain with ten nodes, trigger band, and feedback return.
- P8: evidence-resource column, 3x3 capability matrix, and adaptation side panels.
- P9: three-layer scenario stack and five-step service ladder.

## Icon Library

Lock normal concept icons to `tabler-outline`.

## Header/Footer Defaults

- No page badge.
- No independent footer.
- No confidentiality statement.
- Source/caveat text, if any, must be inline in content areas and traceable to the evidence table.

## Stage Gate

This file only locks the visual system. It does not approve ImageGen blueprint generation.
