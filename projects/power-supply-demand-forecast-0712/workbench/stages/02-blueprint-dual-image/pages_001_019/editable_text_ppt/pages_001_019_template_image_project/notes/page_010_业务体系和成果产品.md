# 业务体系和成果产品

业务体系上，预测对象建议按六类展开，包括需求、供给、平衡、市场、调节、风险。这样可以把传统供需平衡分析与新型电力系统运行中的关键变量放在同一框架下考虑。

运行节奏上，拟覆盖年度、季度、月度、重点时段、月内修正五类。不同节奏对应不同决策需求，年度和季度偏趋势判断，月度和月内修正更强调运行跟踪，重点时段则更关注风险预警。

成果产品上，规划报告、看板、专报、清单、接口、会商助手六类出口。不同产品服务不同使用场景，其中报告和专报更偏正式发布，看板和接口更偏过程支撑，清单和会商助手用于风险跟踪和会商准备。

首期仍建议聚焦月度、季度供需形势分析和年度报告自动化。其他产品和场景可以纳入总体设计，但不作为首期全部交付。
