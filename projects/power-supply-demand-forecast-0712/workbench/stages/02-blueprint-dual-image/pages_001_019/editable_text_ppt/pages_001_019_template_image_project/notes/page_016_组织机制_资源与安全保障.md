# 组织机制、资源与安全保障

组织机制上，建议形成领导统筹、处室牵头、专班推进、专家支撑、生态协同的工作安排。供需预测涉及数据、模型、业务和发布多个环节，需要有清晰的责任链条。

人员配置可按参考口径安排，由 12-18 人逐步增长至 30-50 人。这里是建设和运行所需的参考配置，实际配置还需要结合职责分工、工作量和后续批复情况确定。

预算方面可按精简、中等、全面三档作为参考。不同档位对应的建设范围、平台能力和运行保障强度不同，后续仍需结合首期边界和专项论证进一步细化。

安全边界需要同步纳入，包括数据合规、质量、模型偏差、系统可靠、网络安全和发布审核等方面。相关风险等级和控制措施仍需进一步量化，不能把风险识别等同于风险已经解决。
