# Stage 2 视觉系统锁定：风格05（浅灰白 + 墨绿）

用户已选定风格编号 **05**。以下为视觉系统锁定记录，供全篇18页蓝图与第三阶段还原共同遵守。

## 色板（锁定，不得逐页漂移）

| 用途 | 色值 |
|---|---|
| 页面背景 | `#F2F3EF` |
| 标题 | `#111111` |
| 正文 | `#333333` |
| 次级/说明文字 | `#666666` |
| 线条/分隔线 | `#D7D9D3` |
| 强调色（结论条、图表主色、关键数字、图标） | `#1F5B4D`（墨绿） |
| 风险/警示专用色（非强调色，仅用于红旗/风险条目，避免与墨绿混淆） | `#8B1E1E`（暗砖红，克制使用，仅第16页风险登记表和红旗标记） |

## 页面几何

- 画布：13.333 x 7.5 英寸（16:9）
- 安全边距：0.4 英寸（页面四周不放置文字/关键元素）
- 列网格：12列网格，列间距0.15英寸；内容页常用组合为"左7列主图+右5列侧栏"或"上下两段式"
- 行网格：页眉区（仅章节名，无页码徽章）0.4英寸，正文区至SO WHAT条上沿，SO WHAT/结论收尾条0.5-0.7英寸；不设独立页脚区，安全边距内直接收尾

## Typography Scale（全篇固定15级，见 `ppt-production.md`）

沿用仓库固定 `C0` + `T1-T14`（字号范围见ppt-production.md第623-639行），本项目不新增层级。字体族：中文标题用思源黑体Bold/苹方-简中黑体（或系统等效无衬线粗体），中文正文用思源黑体Regular/苹方-简，英文/数字用Inter或Arial作为备选字体，保证中西文混排时数字不因字体切换产生对齐问题。

## 表面系统（Unified Page Surface System）

- 页面采用**连续纸面**：整页背景统一使用 `#F2F3EF`，不使用大面积纯白卡片作为默认分区语言。
- 内容面板/图表区使用比背景略深或略浅的中性灰面板（面板色阶基于`#F2F3EF`上下浮动，不引入新色相），面板边界用细线`#D7D9D3`或极轻微明暗差，不使用重阴影。
- 结论条/SO WHAT条使用墨绿`#1F5B4D`作为底色，白色或浅米字体，是全篇唯一大面积使用强调色块的区域，用于制造视觉锚点。
- 表格：表头栏使用墨绿浅色调（`#1F5B4D`降低不透明度或浅色调版本）打底，正文区维持连续纸面色，行间用细线分隔，不使用重网格。
- 白色仅作为局部强调（如关键数字底卡或证据标签底），不作为页面主内容底色。

## 图表配色规则

- 主图表使用墨绿`#1F5B4D`为主色，中性灰`#666666`/`#D7D9D3`为对比或背景系列。
- 强调色（墨绿）只用于表达含义：最高优先级数据系列、结论、行动点；不得用于装饰性填充。
- 风险/红旗类图表元素单独使用暗砖红`#8B1E1E`，与墨绿形成"正常=绿、风险=红"的一致语义，全篇仅第16页启用红色系列。

## 页眉页脚系统

- **不设左上角页码徽章**（用户已确认移除）。
- **不设页脚**（用户已确认移除，包括来源说明/证据ID范围/页码等页脚常见内容，均不出现）。
- 页眉仅保留章节名（次级色，T3/T4层级），标注当前所属逻辑段（背景/问题/方案/风控/收束），不含页码徽章。
- 来源说明、证据口径、caveat等原本计划放在页脚的内容，改为随对应证据块/图表就近以小字（T14层级）标注在内容区内部，不单独占用页脚条。
- 章节过渡：无独立章节幕页（18页方案未设章节分隔页），仅用页眉章节名标注逻辑段。
- **全篇任何页面不得出现"内部资料，注意保密""内部文件，请勿外传"等保密声明类文字**（用户已明确要求排除），如需保密声明由用户在使用环节自行处理。

## 圆角/阴影/分隔线/图标

- 圆角：面板与卡片使用小圆角（4-6px等效），不使用大圆角/胶囊形态。
- 阴影：不使用重投影；面板区分主要依赖底色阶差和细线，不依赖阴影堆叠层次。
- 图标库：锁定 `tabler-outline`（细线风格，与"浅灰白+墨绿"的克制调性一致），全篇通用图标不混用其他库；品牌/机构标识（如涉及中电联标识）如需保留原图，按第三阶段图片资产准入门单独判断。

## 密度与留白节奏

- 内容页密度目标：延续Stage 1确认的"高密度"标准——1个结论标题+1主图/框架+2-4证据块+1个含义/行动框+来源脚注。
- 封面（P1）、总结收束页（P18）保持低密度，其余16页保持高密度，不得中途降密度製造"喘息页"。

## 语言判定

- `target_language` = 简体中文（源材料主语言，且未见用户指定其他语言或对话语言冲突，无需额外询问）。
- `language_source` = `source_material`。
- 允许保留原文的专有名词：机构/平台英文名（如EcoVadis、CDP、Achilles、ENR、IFC、OECD等），需登记为 `allowed_foreign_terms`，不译。
