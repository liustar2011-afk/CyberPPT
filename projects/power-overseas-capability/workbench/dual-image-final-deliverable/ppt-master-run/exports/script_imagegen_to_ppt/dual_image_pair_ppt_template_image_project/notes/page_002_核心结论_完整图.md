# 建议由中电联牵头，用'六位一体'体系和四阶段试点，把电力产业链企业出海能力证明从'自证'转向'可信证据'

本页围绕“建议由中电联牵头，用'六位一体'体系和四阶段试点，把电力产业链企业出海能力证明从'自证'转向'可信证据'”展开。
汇报要点：
- "电力产业链企业出海能力证明体系建设已具备推进必要性，应坚持场景牵引、证据支撑、分角色建模、分层产品化、数据化运营和边界清晰原则"
- 1. 补齐企业能力可信表达短板
- 2. 坚持分角色、分场景、分维度、重证据
- 3. 监测是评价有效性的关键支撑
- 4. 服务产品必须进入真实业务流程
- 5. 数据基础设施应服务于场景化能力证明
- 6. 组织实施采用五方协同模式
- 7. 风险边界必须前置控制
- "建议按'规则先行—试点验证—常态运营—规模推广'路径启动首阶段工作"
