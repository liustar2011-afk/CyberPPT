## 3. 内容脑暴：2-3条可选故事线

### 故事线 A（推荐）：完整论证线 —— 为什么做 / 做什么 / 怎么做 / 如何控险

- **核心结论**：全球规则趋严和现有服务空白，使中电联牵头建设"评价+监测+产品+数据+运营+风控"六位一体的行业化能力证明服务体系成为必要且可行的选择，建议按四阶段路径启动试点。
- **Issue tree**：
  1. 外部环境是否已经变化到需要新服务？（Ch1-2：规则趋严、审查方式转变）
  2. 现有市场工具能否替代？（Ch2：五类同类服务逐一强弱对标，均有缺口）
  3. 中电联是否具备牵头资质？（Ch3：组织覆盖、规则承载、数据基础设施先行先试资质）
  4. 能否设计出可执行、风险可控的方案？（Ch3-5、Ch7：六位一体框架、评价方法论、风控机制）
  5. 能否用低成本方式验证可行性？（Ch6：四阶段试点路径与时间线）
- **关键证据ID**：E001-E018（环境背景）、E019-E030（现有服务缺口）、E043-E048（中电联适配性）、E041/E051-E074（六位一体设计）、E086-E096（试点路径）、E098-E105（风控）。
- **最大caveat**：无财务测算数据，无法回答"投入产出比"问题；六位一体设计信息量大，若不精简易导致内容页密度过高、注意力分散。
- **适合受众/场景**：需要"是否批准立项+如何推进"完整决策依据的内部领导层汇报——与本次场景完全匹配。
- **推荐理由**：唯一能同时覆盖"必要性-方案设计-推进路径-风险边界"全部决策要素的故事线，且与原报告结构高度对应，转化风险最低，证据链最完整。

### 故事线 B：评价方法论深挖线 —— 只讲"怎么评"

- **核心结论**：用"四层评价框架+五级结果表达"替代传统总分排名，能更准确反映电力产业链企业的角色与场景差异。
- **关键证据ID**：E018（方法论依据）、E037-E039（角色/能力分类）、E042（四层框架）、E051-E058（方法与结果表达）、E111-E112（指标全表）。
- **最大caveat**：不含组织运营、数据基础设施、风险边界内容，无法回答"要不要投资建设"这一决策问题。
- **适合受众/场景**：面向方法论专家评审会或技术委员会的备案材料，不适合面向决策层的整体方案汇报。
- **不推荐原因**：范围过窄，无法承载本次"内部方案汇报"所需的立项决策信息；其内容将作为故事线A中"评价体系设计"页组的素材来源，而非独立成篇。

### 故事线 C：风控优先线 —— 先讲"我们不做什么"

- **核心结论**：以六不能边界声明和六项控制机制为前提，在风险可控范围内推进评价服务体系建设。
- **关键证据ID**：E049-E050（边界）、E098-E105（风险与控制机制）。
- **最大caveat**：弱化了体系价值主张（为什么需要、能带来什么收益），可能让决策层看到"约束"却看不到"收益"，说服力不足，进取性不够。
- **适合受众/场景**：适合风控/合规部门单独评审风险设计方案。
- **不推荐原因**：作为面向决策层的主线汇报防御性过强，不适合直接作为主叙事；其内容将整合为故事线A收尾前的"风险边界"独立页组。

**推荐**：采用**故事线A**为主线，故事线B、C的核心内容分别整合进故事线A的"评价体系设计"页组和"风险边界"页组。

## 4. SCR 收敛（基于故事线A）

- **Situation**：全球能源转型推动能源投资维持高位（E001-E003），国际采购、融资、供应链和E&S规则正把电力产业链企业出海的"能力证明"要求从资质审查推向持续证据审查（E008-E013）；中国电力产业链企业出海呈现多角色、多场景特征，八类核心能力成为通用需求（E037,E039）。
- **Complication**：企业现有能力表达方式材料分散、口径失真、静态滞后（E019）；国际同类服务——多边行资格预审、供应链平台、ESG评级、工程榜单、E&S融资框架——各有专长也各有结构性缺口，均无法覆盖电力产业链企业的全角色全场景能力证明需求（E021-E030）；若不作为，企业将持续面临"证明成本高、认可度低、风险不可控"的局面。
- **Resolution**：由中电联牵头（具备行业覆盖、规则承载、数据基础设施先行先试资质，E043-E048），建设"评价、监测、产品、数据、运营、风控"六位一体的行业化能力证明服务体系（E041），以四层评价框架+五级结果替代总分排名（E042,E052），以五类场景对应七类服务产品（E056,E060），按"规则先行→试点验证→常态运营→规模推广"四阶段路径推进（首阶段约3个月出规则/产品样板，E086-E090），并以六不能边界声明和六项控制机制前置管控风险（E049,E105）。
