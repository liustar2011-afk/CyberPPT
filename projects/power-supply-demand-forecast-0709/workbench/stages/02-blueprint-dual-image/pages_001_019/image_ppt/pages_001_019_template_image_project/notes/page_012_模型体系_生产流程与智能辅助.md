# 模型体系、生产流程与智能辅助

模型体系不是单一算法选择，而是面向生产运行的组合能力。

在结果一致性和风险表达方面，层级一致性用于保证全国、区域、省级和行业结果可汇总；区间预测和解释性输出用于说明风险边界、变量贡献和预测偏离原因，避免只给出单点结果而缺少解释。

生产流程上，先读取已发布视图和当前修订视图，确认数据版本和指标口径；再执行质量规则和异常检测，生成基础、事件、高频修正和层级汇总特征；随后训练或更新模型，在滚动回测集上生成预测，并执行层级一致性校正。

发布后还要进入误差复盘、漂移监测和再训练触发流程，形成持续改进。

预测成果需要明确数据口径、模型版本、预测区间、假设条件、适用范围和误差说明。智能化能力主要用于问题检索、数据异常提示、预测变化解释、图表生成、报告草拟和会商纪要辅助，不替代人工审校和发布责任。
