# 需请领导审定事项

为推动电力供需形势预测工作由能力建设进入实施推进，提请领导审定或明确有关事项。第一，明确将电力供需形势预测作为中电联行业公共能力建设的重要工作。

第二，明确统计与数智部电力供需分析处牵头，建立跨部门工作专班和会商机制。第三，同意将全国月度与季度供需形势分析、年度报告自动化作为首期工作重点。

第四，同意对现有数据资源、系统能力、授权边界和发布流程开展专项摸底，形成后续论证依据。第五，支持与电网企业、发电集团、电力交易机构、气象机构及有关单位开展数据和业务协同。

第六，原则同意按夯基、成型、扩围、运营四阶段推进有关工作。第七，明确数据安全、网络安全、保密管理、发布审核和成果留痕要求。

预算、人力和采购事项，建议在完成专项论证后，再按有关程序报批。
