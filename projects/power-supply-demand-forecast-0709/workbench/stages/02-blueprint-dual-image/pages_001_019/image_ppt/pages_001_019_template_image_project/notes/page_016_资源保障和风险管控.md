# 资源保障和风险管控

资源保障围绕组织、数据、技术、安全、资金和生态协同展开。组织保障明确牵头处室和工作专班；数据保障推动统计、电网、发电、交易、气象、宏观和新型主体数据分级接入；技术保障统筹存量系统复用和新增平台建设。

安全保障方面，落实分类分级、最小权限、日志审计和应急预案；资金保障按照分阶段建设、分节点验收、分场景扩展安排预算；生态保障方面，与电网企业、发电集团、交易中心、气象机构、科研院所和技术企业建立稳定合作机制。

需要特别说明的是，投资测算和人员配置属于后续专项论证事项。三年新增预算参考档位和人员配置爬坡，仅用于说明资源需求量级，最终预算、人力、采购和外部协同事项，需在数据资源、系统能力和建设边界摸底后，按专项论证、财务批复和有关程序推进。

风险管控重点包括数据合规、数据质量、模型偏差、系统可靠、网络安全、组织协同和成果发布七类风险。相应建立安全合规、月度复盘、季度校模、半年场景增补、年度重构和预测成果分级发布机制，确保建设过程中有边界、有复盘、有调整。
