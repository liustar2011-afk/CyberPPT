# 业务体系和成果产品

业务体系按预测对象、时间尺度和成果产品三条线组织。预测对象覆盖需求侧、供给侧、平衡侧、市场侧、调节侧和风险侧，首期重点围绕全社会用电量、分产业用电量、重点行业用电、最大负荷、装机容量、发电量、利用小时、全国和区域平衡、交易互济及气象扰动，开展月度与季度分析。

需要说明边界的是，新能源有效容量、需求响应、虚拟电厂、储能、充电负荷和市场机制影响，作为后续拓展方向，不在首期一次性全部铺开。这样安排有利于先把高频使用、数据基础相对明确的业务闭环稳定下来。

时间尺度上，建立年度判断、季度校核、月度滚动、重点时段专题、月内修正的体系。年度形成供需总体判断和区域风险提示，季度滚动校核年度判断，月度形成全国总盘、区域和行业分析；迎峰度夏、迎峰度冬和重大事件等重点时段开展专题支撑，月内再依据高频数据修正当月和下月判断。

成果产品上，形成报告、看板、专报、清单、接口、会商助手六类产品。报告包括月度、季度、年度和重点时段专题报告；看板包括领导驾驶舱、全国总盘、区域热力图、指标趋势和风险分布；专报、清单、接口和会商助手分别服务专题分析、风险管理、指标服务和会商支撑，最终形成从研判到发布、再到复盘的业务服务闭环。
