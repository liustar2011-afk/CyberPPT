# 建设原则和首期边界

建设原则上，首先坚持职责牵引，围绕中电联供需形势分析预测、行业发布、政策咨询和会员服务职责开展建设。其次坚持数据可信，优先建立数据目录、指标字典、口径主表、质量规则、授权台账和审计留痕，使预测结果有来源、有版本、有解释。

在运行约束上，坚持安全合规，落实数据安全、网络安全、信息披露和保密管理要求，对敏感数据采用分级授权、脱敏汇聚、最小权限和全链条审计；同时坚持分步实施，按照夯基、成型、扩围、运营四阶段推进。

首期边界建议聚焦一个主闭环、两个支撑能力和三个可验收产品。一个主闭环，是全国月度与季度供需形势分析闭环，覆盖数据接入、质量治理、预测研判、专家审校、报告发布和误差复盘；两个支撑能力，是数据资源目录与指标口径体系、报告自动化与审校发布体系。

三个可验收产品，是月度供需分析报告、季度供需形势分析报告、年度报告自动化原型。首期成果应做到可自动成稿、可人工审校、可版本归档、可回放复盘，后续拓展场景在数据、模型和组织条件具备后再逐步纳入。
